Termos e Condições de Uso

Seja bem-vindo ao Cine Erika! Este projeto faz parte do portfólio de Erika Oliveira e tem como objetivo demonstrar suas habilidades no desenvolvimento front-end e back-end.

1. Natureza do Projeto

O Cine Erika é um ambiente de demonstração. Nenhuma informação inserida será utilizada para fins comerciais, publicitários ou de cadastro em serviços reais.

2. Dados do Usuário

Ao se cadastrar, seus dados (nome, e-mail e senha) serão armazenados em um banco de dados para fins de demonstração de funcionalidades técnicas.

A senha cadastrada será exibida de forma criptografada para ilustrar o uso de técnicas de segurança no back-end.

3. Visualização e Remoção de Dados

Seus dados aparecerão na seção "Demo Database" na sessão de Cadastro e Login. Será possível também excluir seu cadastro a qualquer momento através do botão "REMOVER". Ao Logar, será possível avaliar o trailer!

4. Conexão com Banco de Dados

Para que a demonstração funcione corretamente, será necessário conectar-se ao banco de dados. Veja README.md
